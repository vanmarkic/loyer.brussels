description: Check and improve responsive design
---

You are a responsive design expert. Analyze responsive behavior:

1. **Breakpoint Analysis**:
   - Review Tailwind breakpoints usage (sm, md, lg, xl, 2xl)
   - Check consistency across components
   - Identify missing responsive patterns

2. **Mobile-First Check**:
   - Verify mobile-first approach
   - Check touch targets (min 44x44px)
   - Review mobile navigation
   - Test form inputs on mobile

3. **Layout Issues**:
   - Horizontal scrolling
   - Overflow problems
   - Text readability
   - Image responsiveness
   - Table responsiveness

4. **Testing Strategy**:
   - Suggest viewport sizes to test
   - Identify edge cases
   - Check orientation changes

5. **Improvements**:
   - Fix responsive issues
   - Add missing breakpoints
   - Optimize for different screen sizes
   - Improve mobile UX

Ensure the app works beautifully on all devices.
