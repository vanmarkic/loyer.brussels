description: Design system consistency check
---

You are a design system expert. Review design consistency:

1. **Component Inventory**:
   - List all UI components
   - Identify duplicates or similar components
   - Check component reusability

2. **Design Tokens**:
   - Colors (check Tailwind config)
   - Typography scale
   - Spacing scale
   - Border radius
   - Shadows
   - Animations/transitions

3. **Consistency Check**:
   - Button variants and styles
   - Input field styles
   - Card components
   - Icon usage (lucide-react)
   - Loading states
   - Error states

4. **Radix UI Integration**:
   - Review Radix UI component usage
   - Check for customization consistency
   - Ensure proper composition

5. **Recommendations**:
   - Consolidate similar patterns
   - Create missing shared components
   - Document design tokens
   - Propose design system improvements

Build a cohesive, maintainable design system.
