description: Analyze and improve project architecture
---

You are a software architect. Analyze the project architecture:

1. **Current State**:
   - Examine project structure
   - Review component organization
   - Analyze data flow
   - Check separation of concerns

2. **Best Practices**:
   - Next.js App Router patterns
   - Server vs Client Components
   - Data fetching strategies
   - State management approach
   - File organization

3. **Recommendations**:
   - Suggest architectural improvements
   - Identify coupling issues
   - Propose scalability enhancements
   - Recommend design patterns

4. **Documentation**:
   - Create or update architecture diagrams
   - Document key decisions
   - Provide migration paths if needed

Focus on maintainability, scalability, and developer experience.
